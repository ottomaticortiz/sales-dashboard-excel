# sales-dashboard-excel
This project analyzes simulated retail sales data to evaluate business performance and customer behavior.
## Key Features
- Monthly revenue and customer traffic analysis
- Average transaction value tracking
- Product category performance breakdown
- Day-of-week sales trends
- Interactive dashboard using slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Key Insights
- Vape products generated the majority of revenue
- Sales and customer traffic peaked on weekends
- Average transaction value ranged between $40–$50
